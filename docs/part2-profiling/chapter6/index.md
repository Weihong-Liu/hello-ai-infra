# 第6章 建立你的第一个性能分析报告

## 本章导读

本章综合运用前两章的方法和工具，完成你的第一份性能分析报告。

## 6.1 选择一个慢算子

从 PyTorch 模型中找到一个耗时较高的算子作为分析对象。

## 6.2 运行 baseline

记录当前性能数据作为优化对比的起点。

## 6.3 收集 profiling 数据

使用前面学到的工具采集完整的性能数据。

## 6.4 判断瓶颈类型

根据 profiling 数据判断算子是 Memory-bound 还是 Compute-bound。

## 6.5 提出优化假设

基于瓶颈类型提出 2-3 个可行的优化方向。

编写一份结构化的性能分析报告。
